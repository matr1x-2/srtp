# srtp
本仓库用于存放 srtp 项目“面向GUI智能体的环境注入攻击风险监测与AI安全防护”的代码。

本项目旨在系统分析通用型GUI智能体在对抗环境下对环境注入攻击的脆弱性，并在此基础上设计一种新的防御机制，以提升智能体在复杂交互环境中的安全性与鲁棒性。具体目标包括：构建一套自动化环境注入攻击评估平台，对GUI智能体的安全性进行系统化测试，同时设计与实现针对该类攻击的防御系统，进而形成创新性研究成果，投稿高水平学术论文并参与相关学科竞赛。与现有工作相比，本项目的核心创新点在于：利用大模型的注意力偏移机制进行注入攻击检测，能够高效、精准地定位有害元素，显著提高检测成功率，并在不同场景或界面下具备更强的泛化能力。
