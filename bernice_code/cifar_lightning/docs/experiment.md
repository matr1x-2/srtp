# 实验记录

## 1. 实验目标

在相同的数据增强、优化器和训练轮数设置下，对比不同网络结构在 CIFAR10/CIFAR100 上的分类效果。

## 2. 实验流程

1. 使用 `torchvision.datasets` 自动下载 CIFAR10/CIFAR100。
2. 按固定随机种子将训练集划分为训练集和验证集。
3. 对所有模型使用相同的数据增强策略：
   - 随机裁剪
   - 随机水平翻转
   - 小角度旋转
   - 随机擦除
4. 保持优化器、学习率和训练轮数一致。
5. 以验证集最优模型为准，在测试集上做最终评估。

## 3. 推荐实验顺序

1. 先在 CIFAR10 上运行 `simple_cnn`，确认流程是否正确。
2. 再在 CIFAR10 上测试 `resnet18`，作为强基线。
3. 继续比较 `resnet50`、`vgg16`、`densenet121`。
4. 选出最好模型后，再在 CIFAR100 上重复实验。

## 4. 需要记录的内容

每次实验至少记录：

- 数据集名称
- 模型名称
- 最大训练轮数
- 学习率
- 验证集准确率
- 测试集准确率
- 训练时间
- 是否出现过拟合

## 5. 报告写法模板

你可以按下面的思路写实验过程：

1. 首先使用 `simple_cnn` 验证训练流程是否正常。
2. 然后固定预处理和优化设置，只更换网络结构进行比较。
3. 最后在 CIFAR100 上测试最优模型，观察任务难度提升后的性能变化。
4. 结合训练集和验证集曲线分析是否出现过拟合。
